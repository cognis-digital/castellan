# castellan — Roadmap

## Now (v1.0)
- STRIDE + LINDDUN engine, DREAD risk scoring, attack trees.
- IaC importers: Terraform, CloudFormation, Kubernetes, docker-compose.
- Outputs: table, JSON, SARIF, Markdown, HTML, Mermaid.
- 625 compliance frameworks, 5,000+ threat scenarios, 12,000+ requirements.
- MCP server, CI fail-gate, demo scenarios.

## Next (v1.1)
- More base attack patterns and asset classes.
- Deeper IaC coverage (Bicep, Pulumi, ARM, Helm charts).
- Per-control coverage scoring against a chosen framework profile.

## Later (v2.0)
- Plugin API for custom patterns and importers.
- Diff mode (threat-model deltas between two revisions in CI).
- Pro tier + commercial support (licensing@cognis.digital).
